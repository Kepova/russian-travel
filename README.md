**Описание проекта**

Проект о путешствиях по России, которые многие недооценивают.
Проект содержит галерею красивых мест, подробнее описаны 5 известных мест, путешествие до Байкала на электричках.

**Какие технологии использовали**

* Grid Layout;
* Flex;
* БЭМ-технологии.
